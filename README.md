# Umbrella-SOC---Laboratorio-Wazuh
🛡️ Descripción Umbrella SOC es un script que inicia automáticamente un laboratorio SOC completo basado en Wazuh, levantando **Wazuh Manager**, **Wazuh Indexer** y **Wazuh Dashboard**, y abre el Dashboard en el navegador de manera automática.  Este script permite que con un solo comando (`iniciar umbrella`) tu laboratorio quede 100% operativo
